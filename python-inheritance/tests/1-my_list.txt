======================
How to Use 1-my_list.py
======================

Importing the class:
    >>> MyList = __import__('1-my_list').MyList

Testing basic functionality:
    >>> my_list = MyList()
    >>> my_list
    []
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Testing with negative numbers:
    >>> my_list2 = MyList()
    >>> my_list2.append(-5)
    >>> my_list2.append(0)
    >>> my_list2.append(-10)
    >>> my_list2.print_sorted()
    [-10, -5, 0]

Testing empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []
